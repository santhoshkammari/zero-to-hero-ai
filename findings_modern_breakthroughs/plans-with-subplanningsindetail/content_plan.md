# Content Plan - Modern Breakthroughs in Generative AI

## Running Example: "Creative Studio"
Building a one-person creative studio that can produce a short animated film.
Start with needing a single image → edit it → generate video → add 3D → add audio → full production.

## Concept Ladder
1. The Text-to-Image Revolution (DALL-E, Midjourney, SD evolution) - bedrock
2. How Image Editing Works (inpainting, outpainting, img2img) - building on generation
3. From Still to Motion: Video Generation (Sora, Runway, SVD) - temporal extension
4. Escaping Flatland: 3D Generation (NeRF, Gaussian Splatting, SDS) - spatial extension
5. The Sound of AI: Audio & Music Generation (AudioLM, MusicGen, Bark) - new modality
6. Multimodal Generation: Putting It All Together - convergence
7. The Creative AI Ecosystem & Ethics - implications

## Rest Stop
After section 3 (Video Generation) - reader has a useful mental model of visual generation

## Vulnerability Moments
1. Opening: avoided understanding how these models really differ
2. Video section: still developing intuition for temporal coherence
3. 3D section: Gaussian splatting felt like magic at first
4. Audio section: the tokenization insight was surprising
5. Ethics section: genuinely uncertain about where this leads

## Recurring Analogies
1. Film studio / production pipeline - from still camera → editing suite → video → sound stage
2. Assembly line vs artisan workshop - how generation shifts from craft to engineering
