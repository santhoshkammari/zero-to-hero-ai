# Chapter 06 — Unsupervised Learning

## Status: ✅ v3 COMPLETE — Variable-depth, concept-driven, conversational tone

## Author Intent (cumulative)
1. **When to use what** — clustering vs dim reduction vs anomaly detection, and which algorithm within each
2. **Build real intuition** — PCA finds maximum variance directions, K-means is centroid-based, DBSCAN is density-based. Mental models that stick.
3. **Production-relevant** — sklearn code you can copy into a pipeline, not toy examples
4. **Interview-ready** — EM generalizes K-means, PCA ≈ linear autoencoder, soft vs hard assignments
5. **Modern awareness** — self-supervised learning is how BERT/GPT are pretrained, UMAP > t-SNE for most cases, HDBSCAN > DBSCAN
6. **Variable depth** — Clustering and DimReduction go deep. Anomaly Detection and EM are moderate. Self-supervised is moderate-modern. Association Rules and Topic Modeling are Nice to Know.
