# Emergent Abilities Debate

- Schaeffer et al. (2023): "Are Emergent Abilities a Mirage?"
- Argument: emergence is measurement artifact from discontinuous metrics (exact match)
- Switch to continuous metrics (log-likelihood) → smooth improvement, no phase transition
- Counter: some abilities genuinely absent at small scale (chain-of-thought, in-context learning)
- In-context learning theories:
  1. Implicit gradient descent: attention layers simulate learning algorithm in forward pass
  2. Task location/recognition: model identifies which known behavior to activate
- Both have evidence; practical impact same either way
