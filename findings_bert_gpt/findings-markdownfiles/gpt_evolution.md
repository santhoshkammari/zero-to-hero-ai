# GPT Evolution

- GPT-1 (2018, 117M): Proved pretrain+finetune works. BookCorpus.
- GPT-2 (2019, 1.5B): Zero-shot emerges. "Too dangerous to release." First hints of in-context learning.
- GPT-3 (2020, 175B): Few-shot learning. 300B tokens. $4.6M training. Pattern matching at scale.
- GPT-4 (2023, undisclosed): Multimodal. Passes bar exam. Likely MoE.

Key pattern: each generation unlocked qualitatively new capabilities, not just better performance on old tasks.
